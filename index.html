Assignment Objective,

 

Create an HTML version of your CV/Resume, exactly replicating the format shown in the reference image provided. The goal is to practice the HTML tags and formatting techniques we have learned up to Chapter 4.

 

Guidelines

 

You must use all of the following HTML tags in your assignment:

  • < h1 > to < h6 >
  • < p > (Paragraphs)
  • < b >, < strong >, < i >, < em > (Bold and italic text)
  • < abbr > (Abbreviations)
  • < q > (Short quotitions)
  • < sup >, < sub > (Superscript and subscirpt)
  • < a > (Hyperlinks)
  • < ul >, < ol >, < li > (Lists-unorderlist and ordered)

instruction:
